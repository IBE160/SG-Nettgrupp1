---
story: As a Customer, I want to be able to submit my order for click-and-collect so that the store can prepare my items for pickup.
epic: Ordering
story_points: 8
priority: Must-Have
acceptance_criteria:
  - The checkout page must display a summary of the order.
  - The checkout page must have fields for the customer's name and email address.
  - The system must send an order confirmation email to the customer.
  - The system must send an order notification email to the store.
  - The system must save the order details in the database.
  - The user is redirected to a confirmation page with an order reference number.
  - The checkout page must be responsive and display correctly on mobile, tablet, and desktop devices.
