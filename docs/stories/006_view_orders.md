---
story: As the Business Owner, I want to be able to view all incoming customer orders so that I can manage them effectively.
epic: Admin
story_points: 8
priority: Must-Have
acceptance_criteria:
  - The admin dashboard must display a list of all orders.
  - Each order in the list must display the order ID, customer name, order date/time, and order status.
  - New incoming orders must be highlighted.
  - The orders must be sorted by the most recent first.
  - The admin can click on an order to view its full details.
  - The admin can delete orders that have not been picked up.
  - The order management page must be responsive and display correctly on mobile, tablet, and desktop devices.
