---
story: As the Business Owner, I want to be able to add and manage products in the catalog so that I can keep the inventory up-to-date.
epic: Admin
story_points: 8
priority: Must-Have
acceptance_criteria:
  - The admin can add a new product with a name, description, price, stock quantity, and image.
  - The admin can edit the details of an existing product.
  - The admin can archive a product to remove it from the online store without deleting it from the database.
  - The product management interface must be responsive and display correctly on mobile, tablet, and desktop devices.
